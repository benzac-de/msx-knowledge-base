# Icons

The Material Icons ([https://fonts.google.com/icons?selected=Material+Icons](https://fonts.google.com/icons?selected=Material+Icons)) from Google are integrated in the Media Station X application. All icons are referenced by their icon names. Additionally, each icon can be colored by using the syntax `{COLOR}:{ICON}` (e.g. `msx-red:home`). Please see [Colors](./colors.md) for possible values. A special icon is `blank`. This icon is invisible and can be used as placeholder. Since version **0.1.161**, it is also possible to display the outline version of an icon by using the syntax `outline:{ICON}` or `{COLOR}:outline:{ICON}` (e.g. `outline:home` or `msx-red:outline:home`). Please note that for some icons no outline version is available and they are either displayed unchanged or not at all when using the outline syntax.

## Note on Icon Updates

**Note: The Material Icons have been updated in version 0.1.137 (from v3.0.1 to v4.0.0) and in version 0.1.161 (from v4.0.0 to v9/19/22). Please note that the version v9/19/22 is the final one, because the Material Icons from Google are marked as deprecated and will no longer receive official updates. The follow-up project is the Material Symbols from Google. However, there are currently no plans to integrate these.**

If you would like to search for an icon or to see icon additions/adaptations from version **0.1.137+** and version **0.1.161+** or the outline versions of the icons, please have a look at this link: [https://msx.benzac.de/info/icons.html](https://msx.benzac.de/info/icons.html).
