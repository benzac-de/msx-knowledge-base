# URL Parameters

These parameters are only relevant for the web version of Media Station X (i.e. when using the URL `http://msx.benzac.de` or `https://msx.benzac.de`) and can be used to override some application settings with fixed values and to set some platform-specific functions.

**Note: If you set some application settings via these parameters, the corresponding settings in the application are disabled. For example, if you set the start parameter, the start parameter in the application settings is disabled.**

## Parameter Syntax

Parameter syntax of URL parameters.

| Parameter | Type | Example | Description |
|---|---|---|---|
| `animate` | `number` | `animate=0`<br>`animate=1`<br>`animate=2` | Sets the animation settings.<br><br>- `0`: Off<br>- `1`: JS<br>- `2`: CSS (default for most platforms) |
| `back` | `number` | `back=0`<br>`back=1` | Sets the back function.<br><br>- `0`: Off (use back key only; default for most platforms)<br>- `1`: On (use web history) |
| `background` | `number` | `background=0`<br>`background=1` | Sets the background function.<br><br>- `0`: Off (transparant)<br>- `1`: On (opaque; default for most platforms) |
| `busy` | `number` | `busy=0`<br>`busy=1`<br>`busy=2` | Sets the busy function (for data and video requests).<br><br>- `0`: Off (no busy indicator)<br>- `1`: Delayed busy indicator (default for most platforms)<br>- `2`: Instant busy indicator |
| `caption` | `number` | `caption=0`<br>`caption=1`<br>`caption=2` | Sets the caption settings (visibility of the options/menu caption that is indicated in the lower right corner of the content screen).<br><br>- `0`: Off (options/menu caption is hidden)<br>- `1`: Visible if available (options/menu caption is visible if the menu button is available)<br>- `2`: Always visible (default for most platforms)<br>- `3`: Always visible (will be set automatically if the value was `1` and the menu button was pressed) |
| `center` | `number` | `center=0`<br>`center=1` | Sets the center function.<br><br>- `0`: Off (UI is aligned at the top left)<br>- `1`: On (UI is aligned in the center; default for most platforms) |
| `dictionary` | `string` | `dictionary=http://msx.benzac.de/dic/en.json`<br>`dictionary=none`<br>`dictionary=custom` | Sets the dictionary. This parameter is designed for testing purposes or special use cases. If this parameter is to `none`, no dictionary will be used (even if the startup data contains one or a custom dictionary is set). If this parameter is to `custom`, the custom dictionary will be used in preference (even if a `start` parameter is set). For final usage, please set the URL as `dictionary` property in the menu or content data or use the `dictionary:{URL}` action. For more information, please see [Dictionary Structure](./dictionary-structure.md). |
| `exit` | `number` | `exit=0`<br>`exit=1`<br>`exit=2`<br>`exit=3`<br>`exit=4`<br>`exit=11`<br>`exit=12`<br>`exit=13`<br>`exit=14` | Sets the exit function.<br><br>- `0`: Off (exit function is hidden)<br>- `1`: Platform-specific (default for most platforms)<br>- `2`: Perform web history back<br>- `3`: Perform web history start<br>- `4`: Perform window close<br>- `11`: Platform-specific (with direct execution during back navigation)<br>- `12`: Perform web history back (with direct execution during back navigation)<br>- `13`: Perform web history start (with direct execution during back navigation)<br>- `14`: Perform window close (with direct execution during back navigation)<br><br>**Note: Direct execution during back navigation means that the assigned action is executed immediately when the user presses the back button while no further navigation step back is possible. By default, the context menu is displayed in this case and no direct execution is performed.** |
| `fullscreen` | `number` | `fullscreen=0`<br>`fullscreen=1`<br>`fullscreen=2` | Sets the fullscreen function.<br><br>- `0`: Off (fullscreen is not available)<br>- `1`: On (fullscreen is always active and cannot be changed; default for most platforms)<br>- `2`: Toggle (fullscreen can be toggled) |
| `init` | `number` | `init=0`<br>`init=1`<br>`init=2` | Sets the init value.<br><br>- `0`: Default (default for most platforms)<br>- `1`: Ignore stored content state<br>- `2`: Enforce use of stored content state (by default, the stored content state is not used if a start parameter is set) |
| `input` | `number` | `input=0`<br>`input=1`<br>`input=2`<br>`input=3`<br>`input=4` | Sets the input settings.<br><br>- `0`: Off<br>- `1`: Remote Only<br>- `2`: Click & Swipe<br>- `3`: Drag & Drop (default for most platforms)<br>- `4`: Move & Execute |
| `layout` | `number`\|`string` | `layout=0.25`<br>`layout=0.33`<br>`layout=0.5`<br>`layout=0.66`<br>`layout=0.75`<br>`layout=0.8`<br>`layout=0.9`<br>`layout=1`<br>`layout=1.5`<br>`layout=3`<br>`layout=6`<br>`layout=180p`<br>`layout=240p`<br>`layout=360p`<br>`layout=480p`<br>`layout=540p`<br>`layout=576p`<br>`layout=648p`<br>`layout=720p`<br>`layout=1080p`<br>`layout=2160p`<br>`layout=4320p`<br>`layout=auto`<br>`layout=detect` | Sets the layout settings.<br><br>- `0.25`\|`180p`: 180p<br>- `0.33`\|`240p`: 240p<br>- `0.5`\|`360p`: 360p<br>- `0.66`\|`480p`: 480p<br>- `0.75`\|`540p`\|`sd`: 540p<br>- `0.8`\|`576p`: 576p<br>- `0.9`\|`648p`: 648p<br>- `1`\|`720p`\|`hd`: 720p<br>- `1.5`\|`1080p`\|`shd`: 1080p<br>- `3`\|`2160p`\|`uhd`\|`4k`: 2160p<br>- `6`\|`4320p`\|`8k`: 4320p<br>- `auto`: Auto detect (with resize adjustment)<br>- `detect`: Auto detect (without resize adjustment) |
| `leave` | `number` | `leave=0`<br>`leave=1` | Sets the leave function. The leave function performs a web history back on execution and can be used if the Media Station X application is opened from another application (or from the local Media Station X application).<br><br>- `0`: Off (leave function is hidden)<br>- `1`: On (leave function is visible) |
| `platform` | `string` | `platform=default`<br>`platform=tizen`<br>`platform=samsung`<br>`platform=lg`<br>`platform=netcast`<br>`platform=panasonic`<br>`platform=chromecast`<br>`platform=ios`<br>`platform=android`<br>`platform=firetv`<br>`platform=hbbtv`<br>`platform=hbbtvop`<br>`platform=uwp`<br>`platform=mac`<br>`platform=win` | Sets the platform identifier (usually this is not set by parameter, because each platform has its own index file). |
| `playback` | `number` | `playback=0`<br>`playback=1`<br>`playback=2`<br>`playback=3` | Sets the playback mode.<br><br>- `0`: Stop (video/audio stops when the application goes to sleep)<br>- `1`: Pause only (video/audio pauses when the application goes to sleep and stays in that state when the application wakes up)<br>- `2`: Pause/Play (video/audio pauses when the application goes to sleep and continues to play when the application wakes up)<br>- `3`: Keep playing (video/audio continues to play when the application goes to sleep or the system controls the behavior; default for most platforms) |
| `pointer` | `number` | `pointer=0`<br>`pointer=1`<br>`pointer=2`<br>`pointer=3`<br>`pointer=4`<br>`pointer=5` | Sets the pointer mode.<br><br>- `0`: Off (no mouse and touch events are handled)<br>- `1`: Mouse (only mouse events are handled)<br>- `2`: Touch (only touch events are handled)<br>- `3`: Merge (mouse and touch events are handled together)<br>- `4`: Switch (either mouse or touch events are handled, whichever is detected first; default for most platforms)<br>- `5`: Combi (only passive mouse, but all touch events are handled) |
| `remote` | `number` | `remote=0`<br>`remote=1`<br>`remote=2`<br>`remote=3`<br>`remote=4` | Sets the remote settings.<br><br>- `0`: Off<br>- `1`: Minimalistic (default for most platforms)<br>- `2`: Navigation Frame<br>- `3`: Corner Controls<br>- `4`: Complex |
| `scale` | `string` | `scale=default`<br>`scale=auto`<br>`scale=device`<br>`scale=0.75`<br>`scale=fix:0.75` | Sets the scale settings (viewport settings).<br><br>- `default`: No scaling (default for most platforms)<br>- `auto`: Auto detect (with resize adjustment)<br>- `device`: Set scale factor to device width (not fixed)<br>- `{NUMBER}`: Set scale factor to specific number (not fixed)<br>- `fix:{NUMBER}`: Set scale factor to specific number (fixed) |
| `secure` | `number` | `secure=0`<br>`secure=1`<br>`secure=2`<br>`secure=3`<br>`secure=4` | Sets the secure settings (protocol updates from `http://` to `https://`).<br><br>- `0`: Update JSON files and plugins in a secure context and never update media files (default for most platforms)<br>- `1`: Update JSON files, plugins, and media files in a secure context<br>- `2`: Force update of JSON files and plugins and never update media files<br>- `3`: Force update of media files and update JSON files and plugins in a secure context<br>- `4`: Force update of JSON files, plugins, and media files |
| `speed` | `number` | `speed=0`<br>`speed=1`<br>`speed=2`<br>`speed=3` | Sets the playback speed function.<br><br>- `0`: Off (speed button is always disabled)<br>- `1`: Video only (speed button is only enabled for videos)<br>- `2`: Audio only (speed button is only enabled for audios)<br>- `3`: Video & Audio (speed button is always enabled; default for most platforms) |
| `start` | `string` | `start=menu:user:http://sc.msx.benzac.de/msx/service.php`<br>`start=menu:user:http://sc.msx.benzac.de/msx/service.php>index:2`<br>`start=menu:user:http://sc.msx.benzac.de/msx/service.php>index:2>index:4`<br>`start=menu:user:http://sc.msx.benzac.de/msx/service.php>index:2>index:4>execute` | Sets the start parameter. The focus separator `>` can be used to focus (and execute) an item on load. |
| `suspend` | `number` | `suspend=0`<br>`suspend=1`<br>`suspend=2` | Sets the suspend mode.<br><br>- `0`: Off (application will never suspend/resume)<br>- `1`: Visibility handler (document visibility is checked to suspend/resume the application)<br>- `2`: System handler (system events are checked to suspend/resume the application; default for most platforms) |
| `t` | `number` | `t=1542793102320` | Sets the current timestamp (difference to the start timestamp must be less than 1 second to skip the splash screen). |
| `transform` | `number` | `transform=0`<br>`transform=1`<br>`transform=2` | Sets the transformation settings.<br><br>- `0`: Off<br>- `1`: 2D (default for most platforms)<br>- `2`: 3D |
| `volume` | `number` | `volume=0`<br>`volume=1`<br>`volume=2` | Sets the volume function.<br><br>- `0`: Off (volume function is hidden)<br>- `1`: On (default for most platforms)<br>- `2`: Mute only (volume function is visible, but can only be used to toggle the muted state) |
| `welcome` | `string` | `welcome=menu:user:http://sc.msx.benzac.de/msx/service.php`<br>`welcome=menu:user:http://sc.msx.benzac.de/msx/service.php>index:2`<br>`welcome=menu:user:http://sc.msx.benzac.de/msx/service.php>index:2>index:4`<br>`welcome=menu:user:http://sc.msx.benzac.de/msx/service.php>index:2>index:4>execute` | Sets the welcome pages that are used if no start parameter is set. The focus separator `>` can be used to focus (and execute) an item on load.<br><br>**Note: This parameter will override the welcome pages in the application settings, however, it will not disable the function to setup the start parameter (and to setup welcome pages via the `welcome` property).** |
| `zoom` | `string` | `zoom=default`<br>`zoom=auto`<br>`zoom=0.75` | Sets the zoom settings.<br><br>- `default`: No zooming (default for most platforms)<br>- `auto`: Auto detect (with resize adjustment)<br>- `{NUMBER}`: Set zoom factor to specific number |
